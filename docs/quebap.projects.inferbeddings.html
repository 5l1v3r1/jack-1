quebap.projects.inferbeddings.sampling.random_truncated_exponential(w, num_samples)¶

Sample from truncated independent exponential distributions using weights w (or inverse exponential distribution for positive weight components). :param w: a [num_rows, dim] matrix of num_rows weight vectors. :param num_samples: the number of samples to produce.

Returns:x,p where x is a tensor [num_samples, num_rows, dim] of samples and p is a matrix [num_samples, num_rows] of the probabilities of each sample with respect to each row.
